# 开发记录 / Changelog

## v2.2 (2026-03-12) - 学校机房重构

### 本次变更
- **默认开启删除源文件** (`DELETE_AFTER_MERGE=true`)
- 更新使用场景说明（24h自动化、夜间上传、防限速）

### 已修复问题
1. ✅ **concat 文件格式** - 与 Go 源项目一致（无引号、相对路径）
2. ✅ **ffprobe 视频验证** - 压缩后验证视频有效性
3. ✅ **目录删除** - 使用 `shutil.rmtree` 替代 `rmdir`
4. ✅ **资源限制** - docker-compose 添加 CPU/内存限制
5. ✅ **换行符统一** - 全部改为 Unix (LF) 格式

### 测试记录
- **测试时间**: 2026-03-12 学校机房
- **测试环境**: Intel i5-10400 / 8GB / Windows
- **测试数据**: 60个小米摄像头片段 (747MB)
- **合并结果**: ✅ 3.8秒完成，输出 747.6MB MOV
- **压缩测试**: ⏱️ H.265编码较慢（正常，i5-10400性能限制）
