# NAS 工具集

个人 NAS 自动化工具集合，适用于飞牛/N100/8845 mini PC 等设备。

⚠️ **开发阶段**：代码正在测试，生产环境请谨慎使用。

## 项目列表

### 1. WebDAV 限速上传器 v2.1

单线程限速上传工具，适合夜间低带宽上传。

**功能：**
- 单线程文件上传
- 令牌桶限速（默认 1MB/s）
- MD5 校验
- SQLite 状态持久化
- 1小时轮询
- 上传后自动删除本地文件

**目录：** `webdav-uploader/`

### 2. 小米视频处理工具 v2.1

小米摄像头视频自动合并+压缩。

**功能：**
- 合并 1分钟片段 → 小时级 MOV
- MOV → MKV (H.265) 压缩
- 断点续传（SQLite 状态）
- 单容器处理完即退出

**目录：** `xiaomi-video/`

## 快速开始

```bash
cd webdav-uploader  # 或 xiaomi-video
cp config.yaml.example config.yaml
# 编辑配置
docker-compose up -d
```

## 硬件建议

- **CPU**：视频压缩建议 4核+（H.265 吃 CPU）
- **内存**：8GB+（大文件处理）
- **磁盘**：SSD 可显著提升处理速度
- **网络**：上传工具建议夜间低峰期运行

## 已知问题

- 视频压缩在低端 CPU 上较慢（可改用 x264）
- 限速器在极小带宽场景需调整参数
